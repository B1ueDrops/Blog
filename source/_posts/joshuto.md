## joshuto键位配置

| 键位     | 功能               |
| -------- | ------------------ |
| `ctrl+g` | 启动lazygit        |
| `ctrl+c` | 使用git clone      |
| `ctrl+f` | 使用fzf搜索文件    |
| `ctrl+s` | 使用zoxide进入目录 |
| `gd`     | 去Downloads目录    |
| `gc`     | 去`.config`目录    |
| `ctrl+b` | `cd -`             |
