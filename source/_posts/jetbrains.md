## 功能面板

Jetbrains中有不同的功能面板, 使用如下快捷键可以toggle这些面板, 并且切换Focus.

| 功能面板              | 快捷键  |
| --------------------- | ------- |
| Explorer              | cmd + 1 |
| Run Console           | cmd + 4 |
| Debug Console         | cmd + 5 |
| Diagnostic Console    | cmd + 6 |
| Git View Console      | cmd + 9 |
| Git Operation Console | cmd + 0 |
